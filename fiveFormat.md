# 문장의 5형식
영어 문장을 이루는 가장 기본적인 골격을 배울거에요.<br>
모든 문장은 결국 이 다섯 가지 형태로 나뉩니다.

|형식|구성요소|
|:------:|:------:|
|1형식| S + V|
|2형식| S + V + C|
3형식| S + V + O
4형식| S + V + I.O + D.O
5형식| S + V + O + O.C
## 1형식(S + V)
주어 + 동사
## 2형식(S + V + C)
주어 + 동사 + 보어
## 3형식(S + V + O)
주어 + 동사 + 목적어
## 4형식(S + V + I.O + D.O)
주어 + 동사 + 간접목적어 + 직접목적어
## 5형식(S + V + O + O.C)
주어 + 동사 + 목적어 + 목적격 보어
